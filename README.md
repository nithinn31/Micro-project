## Title
Student Career Recommendation System using Artificial Neural Network

## Introduction
This project uses an Artificial Neural Network (ANN) to suggest suitable career options for students based on their academic performance and skills. It helps students choose the right career path using machine learning.

## Objectives
To build a career recommendation system

To analyze student academic and skill data

To predict suitable career options

To evaluate model performance

## Tools & Technologies
Python

Pandas, NumPy

Scikit-learn

TensorFlow / Keras

## Dataset
The dataset contains:

Student marks

Skill levels

Career labels (target output)

## Methodology
Data preprocessing

Feature selection

ANN model building

Model training

Prediction and evaluation

## Evaluation Metrics
Accuracy

Precision

Recall

F1-score

## Result
The ANN model provides accurate career predictions based on student data.

## Conclusion
This system helps students make better career decisions using AI and improves guidance accuracy.
